# Replacer
A cli tool for replacing strings in files or texts. It should work on Windows, Mac, and Linux.
It replaces any text with the one specified. Works on txt and csv files, as well as straight in the command promt. 
---
# Usage
 There are two usage for this tool:
 1. It replaces text in file by using the `-f` flag and specifying the file.

 2. It replaces and prints the text directly using the `-t` flag.
---
# Installation

1. Clone this github directory or download it onto your machine either by downloading zip file or using the command: 
```
git clone https://github.com/mishilen/Text-Replacer
```
2. Go into the folder for this directory. You can use the command:
```
cd Text-Replacer
```
3. In that directory, use the command
```pip install -e .```

4. Use the command `replacer -help` to get more information on the usage.
